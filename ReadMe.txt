Introduction
-------------
UST-Test Automation framework is a hybrid framework,it supports multiple data layer(DB,Excel,CSV,XML).


Installation
------------
1.JDK 1.7 or Above
2.Eclipse IDE 4.X 
3.Install TESTNG Plugins on eclipse.

Configuration 
-------------
1.Open the Eclipse IDE.
2.Navigate to File Menu->select import option
3.Choose the General option.
4.Select Existing project int workspace.
5.Select the location of your project path.
6.Click Finish.
